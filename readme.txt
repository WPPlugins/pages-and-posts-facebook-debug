== Description ==

* **English** : Add a widget on the Wordpress Dashboard that let you debug all your URLs on the Facebook Debugguer tool to scrape new informations. If you have a problem, you can <a href="http://www.eralion.com/contactez-nous/" target="_blank">contact-us</a>.

* **Français** : Ajoute un widget sur le tableau de bord Wordpress qui vous permet d'envoyer toutes les adresses de vos pages et articles vers l'outil de debug Facebook afin de mettre à jour les nouvelles informations. Si vous rencontrez un soucis, vous pouvez <a href="http://www.eralion.com/contactez-nous/" target="_blank">nous contacter</a>.


== Installation ==

* **English** :
1. Download pagesandpostsfacebookdebugguer.zip from the "download" link on the web page where you're viewing this.
2. Decompress the file contents.
3. Upload the pagesandpostsfacebookdebugguer folder to your WordPress plugins directory (/wp-content/plugins/).
4. Activate the Facebook Debug plugin from the WordPress back-office.

* **Français** :
1. Téléchargez pagesandpostsfacebookdebugguer.zip sur le lien de téléchargement sur cette même page.
2. Décompressez l'archive téléchargée.
3. Mettez en ligne le dossier pagesandpostsfacebookdebugguer dans votre dossier de plugins (/wp-content/plugins/).
4. Activez le plugin Facebook Debug depuis les extentions.

== Usage ==

* **English** : Go to your Dashboard, click on Facebook Debug blue button and wait until it finishs.
* **Français** : Depuis votre tableau de bord, cliquez sur le bouton bleu du plugin Facebook Debug et patientez jusqu'a ce que l'opération soit terminée.
